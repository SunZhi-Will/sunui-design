# Sun UI Design System

歡迎使用 Sun UI Design System，這是一個現代化的 React 組件庫。

## 組件列表

我們的組件庫包含以下主要組件：

- Card - 卡片組件
- FloatingButton - 浮動按鈕
- SocialButton - 社交媒體按鈕
- SidePanel - 側邊面板
- FileUpload - 文件上傳
- FilterGrid - 過濾網格
- GradientBackground - 漸變背景

## 使用方法

要使用這些組件，您需要先安裝相關的包：

```bash
npm install @sunui-design/core
```

然後在您的代碼中導入需要的組件：

```jsx
import { Card } from '@sunui-design/core';
```

## 主題定制

Sun UI 支持主題定制，您可以通過配置來改變組件的外觀。
